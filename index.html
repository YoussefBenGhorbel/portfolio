Enterprise Network Infrastructure Design & Implementation

Academic Integration Project

Designed, configured, and simulated a complete and secure network infrastructure for a fictional two-site enterprise. This project showcases end-to-end network engineering skills from planning and IP addressing to security policy implementation and high-availability configuration using Huawei equipment.

Key Features

  • Hierarchical multi-site network topology.
  • VLANs for traffic segmentation and security.
  • Huawei firewalls with DMZ, Trust, and Untrust zones.
  • High availability with VRRP & HRP for failover.
  • Centralized Wi-Fi management using CAPWAP.

Technologies Used

  • Huawei Enterprise Switches & Routers
  • Huawei USG Firewalls
  • Huawei eNSP Simulator
  • SSL VPN for Secure Remote Access
  • Link Aggregation (Eth-Trunk/LACP)
  • Static/Dynamic Routing, NAT, ACLs
